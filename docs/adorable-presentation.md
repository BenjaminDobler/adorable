# Adorable - AI-Powered Angular IDE

## Slide 1: Title

# Adorable

### AI-Powered Full-Stack Angular IDE

*Build applications using natural language*

---

## Slide 2: What is Adorable?

An **AI-native IDE** that transforms how developers build Angular applications

| The Problem | The Solution |
|-------------|--------------|
| Traditional coding is time-consuming | Describe apps in plain English |
| Design-to-code gap is painful | AI generates complete file structures |
| Setup and configuration overhead | Instant preview and deployment |

---

## Slide 3: Key Features

- **Natural Language Coding** - Describe what you want, AI writes the code
- **Figma Integration** - Import designs and auto-generate components
- **Dual Runtime** - Browser (WebContainer) or Local (Docker)
- **Visual Inspector** - Click elements to identify and edit components
- **Annotation Overlay** - Draw on preview to communicate design intent
- **Time Travel** - Restore any previous state from chat history

---

## Slide 4: Architecture

| Frontend | Backend |
|----------|---------|
| Angular 21 with Signals | Node.js / Express |
| Monaco Editor | Prisma + SQLite |
| Real-time preview with HMR | Docker Manager |
| Smart Container Engine | AI Providers (Claude, Gemini) |

---

## Slide 5: What Makes It Unique

- **Agent Loop** - Multi-turn AI interaction with streaming code generation
- **Progressive Updates** - Files stream in real-time with immediate HMR
- **Skills System** - Pluggable specialized AI capabilities
- **Screenshot Tool** - AI can visually verify its own work
- **VS Code Sync** - Live file sync between IDE and VS Code

---

## Slide 6: The Workflow

```
[Describe App] → [AI Generates Code] → [Live Preview] → [Visual Feedback] → [AI Refines] → [Deploy]
```

**From idea to deployed app in minutes, not hours**
