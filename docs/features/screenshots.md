# Screenshots

Capture a region of the preview to use as visual context in your AI conversation.

## Taking a Screenshot

1. Click the **Screenshot** button (camera icon) in the preview toolbar
2. Click and drag to select a rectangular region on the preview
3. The captured image is attached to your next chat message

## Usage

Screenshots are useful for:

- Showing the AI exactly which part of the UI you're talking about
- Capturing a visual bug to describe alongside your instructions
- Providing before/after context when iterating on designs
