# Desktop App

The Adorable desktop app (macOS) runs the editor locally with additional features not available in the cloud version.

## Desktop-Only Features

### Undock Preview

Click the **Undock** button in the preview toolbar to pop the preview out into a separate window. This is useful for multi-monitor setups — put the preview on one screen and the editor on another.

### DevTools

Click the **DevTools** button (code brackets icon) to open browser developer tools for the preview. This gives you the full Chrome DevTools experience (Elements, Console, Network, etc.) for debugging your app.

### Local Containers

The desktop app can run your project in a local Docker container, giving you full filesystem access and native performance without cloud dependencies.

## Installation

Download the macOS app from the releases page. The desktop app bundles both the client and server — no separate setup needed.
