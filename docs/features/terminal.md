# Terminal

The built-in terminal gives you command-line access to your project.

## Opening the Terminal

Click the **Terminal** tab in the left sidebar.

## Common Uses

- Install npm packages: `npm install some-package`
- Run scripts: `npm run build`
- Check logs or debug output
- Run Angular CLI commands: `ng generate component my-component`

## AI and Terminal

The AI can also run terminal commands as part of its work — for example, installing a package it needs or running a build to check for errors. These commands appear in the terminal output.
