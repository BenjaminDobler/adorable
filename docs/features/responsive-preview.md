# Responsive Preview

Preview your app at different screen sizes to test responsive layouts.

## Device Presets

The preview toolbar includes three device buttons:

| Device | Width | Icon |
|--------|-------|------|
| Phone | 375px | Small rectangle |
| Tablet | 768px | Medium rectangle |
| Desktop | 100% (fills available space) | Monitor |

Click any device button to resize the preview. The active device is highlighted.

## Fullscreen Mode

Click the **Fullscreen** button (expand icon) to hide the sidebar and give the preview the full window width. Click again to restore the sidebar layout.

## Tips

- Use responsive preview together with the AI: "Make the navigation collapse into a hamburger menu on mobile"
- The Inspector, Measure, and Annotation tools all work at any preview size
- Device switching is instant — no reload needed
