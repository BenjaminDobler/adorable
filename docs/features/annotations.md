# Annotations

Adorable offers two annotation modes for communicating visual changes to the AI: freehand annotation and multi-element annotation.

## Freehand Annotation

Click the **Annotate** button (pencil icon) in the preview toolbar.

- Draw directly on the preview to circle, underline, or point at areas you want changed
- The annotation is captured as a screenshot and sent to the AI along with your chat message
- Great for pointing out visual issues like "move this button to the right" or "make this section wider"

## Multi-Element Annotation

Click the **Multi-Annotate** button (cursor with plus icon) in the preview toolbar.

### Adding Elements

1. Click elements in the preview — each gets a numbered purple badge
2. A panel opens on the left showing all selected elements
3. Add a note to each element describing what you want changed (e.g., "make this bold", "change color to red")

### Managing Annotations

- **Click a badge** on the preview to jump to that item in the panel
- **Remove** individual items from the panel
- **Clear all** to start over
- Notes update live — type in the panel and the badge tooltip updates in the preview

### Sending to AI

When you're done annotating, type your overall instruction in the chat and send. The AI receives all annotated elements with their notes and makes the changes.

## When to Use Which

| Mode | Best For |
|------|----------|
| Freehand | Pointing at visual areas, layout issues, drawing arrows to show movement |
| Multi-Element | Precise element-by-element changes, batch style updates, renaming text across multiple elements |
