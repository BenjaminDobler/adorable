# File Explorer

The File Explorer lets you browse, open, and manage all files in your project.

## Opening Files

Click any file in the tree to open it in the code editor. The editor appears above or beside the preview depending on your layout.

## File Operations

Right-click a file or folder for options:

- **Create** new files or folders
- **Rename** existing files
- **Delete** files or folders
- **Upload** files from your computer

## Integration with AI

When you open a file in the editor, the AI can see which file you're looking at. You can reference it in chat: "In the file I have open, change the title text."
