# Figma Integration

Import designs from Figma directly into Adorable to turn them into working Angular components.

## Setup

Install the Adorable Figma plugin from the Figma Community. The plugin lets you export selected frames or components to your Adorable project.

## Importing a Design

1. In Figma, select the frame or component you want to export
2. Run the Adorable plugin and click **Export**
3. In Adorable, open the **Figma** tab in the left sidebar
4. The imported design appears with its layout, styles, and assets
5. The AI converts the design into Angular components

## What Gets Imported

- Layout structure (frames, auto-layout, constraints)
- Colors, typography, and spacing
- Images and icons (as assets)
- Component hierarchy

## Tips

- Export individual sections rather than entire pages for better results
- The AI works best with well-organized Figma files that use auto-layout
- After import, you can use the chat to refine the generated code
