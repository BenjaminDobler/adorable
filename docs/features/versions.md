# Versions

The Versions panel lets you travel back in time through your project's history.

## How It Works

Every AI message creates a snapshot of all project files. The Versions panel shows a timeline of these snapshots.

## Restoring a Version

1. Click the **Versions** tab in the left sidebar
2. Browse the list of past messages/snapshots
3. Click a version to restore your project files to that point in time

## Use Cases

- **Undo a bad change:** If the AI made changes you don't like, go back to a previous version
- **Compare approaches:** Try different instructions and switch between the results
- **Recovery:** Accidentally deleted something? Restore from a snapshot before it was deleted
