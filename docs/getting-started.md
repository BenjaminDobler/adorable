# Getting Started

## Creating Your First Project

1. Sign in to Adorable
2. Click **New Project** on the dashboard
3. Choose a starter template or start from scratch
4. The editor opens with your project's preview on the right and the AI chat on the left

## The Editor Layout

The editor has three main areas:

- **Left sidebar** — Switch between Chat, Terminal, Files, Figma, Versions, Insights, Translations, and Settings tabs
- **Preview pane** — Live preview of your running app with a toolbar along the top
- **Code editor** — Opens when you select a file, showing the source code

## Building with AI

Type what you want in the chat panel. For example:

- "Add a navigation bar with Home, About, and Contact links"
- "Change the background color to dark blue"
- "Create a login form with email and password fields"

The AI will modify your project files and the preview updates in real time.

## Configuring Your AI Provider

1. Click your avatar in the top-left corner
2. Go to **Profile**
3. Add your API key for Anthropic or Google Gemini
4. Select your preferred model

## Preview Toolbar

The toolbar above the preview gives you access to:

| Button | Description |
|--------|-------------|
| Device icons | Switch between phone, tablet, and desktop preview sizes |
| Inspect | Click elements to view their properties and styles |
| Measure | Measure distances and visualize layouts between elements |
| Screenshot | Capture a region of the preview |
| Annotate | Draw on the preview to point out changes you want |
| Multi-Annotate | Select multiple elements and add notes to each |
| Refresh | Reload the preview |
| Fullscreen | Expand the preview to fill the screen |
