Die Desktop-Anwendung hat Zugriff auf das lokale Dateisystem und kann Shell-Befehle ausfuehren (z.B. npm install, ng serve). Diese Befehle werden vom KI-Agenten im Rahmen der Projektgenerierung ausgefuehrt. Der Benutzer ist dafuer verantwortlich, KI-generierte Aktionen vor der Ausfuehrung zu ueberpruefen.

The desktop application has access to the local file system and can execute shell commands (e.g., npm install, ng serve). These commands are executed by the AI agent as part of project generation. The user is responsible for reviewing AI-generated actions before execution.
