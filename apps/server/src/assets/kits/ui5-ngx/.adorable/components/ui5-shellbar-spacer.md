# ShellBarSpacer

**Type:** Component
**Selector:** `<ui5-shellbar-spacer>`
**Import:** `import { ShellBarSpacerComponent } from '@ui5/webcomponents-ngx/fiori/shell-bar-spacer';`
**Export As:** `ui5ShellbarSpacer`
**Package:** `@ui5/webcomponents-fiori` (fiori)

## Basic Usage
```html
<ui5-shellbar-spacer></ui5-shellbar-spacer>
```

## Description
The ui5-shellbar-spacer is an element, used for visual separation between the two content parts of the ui5-shellbar. **Note:** The ui5-shellbar-spacer component is in an experimental state and is a subject to change.
