# MenuSeparator

**Type:** Component
**Selector:** `<ui5-menu-separator>`
**Import:** `import { MenuSeparatorComponent } from '@ui5/webcomponents-ngx/main/menu-separator';`
**Export As:** `ui5MenuSeparator`
**Package:** `@ui5/webcomponents` (main)

## Basic Usage
```html
<ui5-menu-separator></ui5-menu-separator>
```

## Description
The ui5-menu-separator represents a horizontal line to separate menu items inside a ui5-menu.
