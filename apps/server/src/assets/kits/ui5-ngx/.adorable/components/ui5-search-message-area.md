# SearchMessageArea

**Type:** Component
**Selector:** `<ui5-search-message-area>`
**Import:** `import { SearchMessageAreaComponent } from '@ui5/webcomponents-ngx/fiori/search-message-area';`
**Export As:** `ui5SearchMessageArea`
**Package:** `@ui5/webcomponents-fiori` (fiori)

## Basic Usage
```html
<ui5-search-message-area [text]="..."></ui5-search-message-area>
```

## Description
import "@ui5/webcomponents-fiori/dist/SearchMessageArea.js";

## Inputs
| Name | Type | Default | Description |
|------|------|---------|-------------|
| `text` | `string | undefined` | `undefined` | Defines the text to be displayed. |
| `description` | `string | undefined` | `undefined` | Defines the description text to be displayed. |

> **IMPORTANT:** Use property bindings `[inputName]="value"`. Do NOT use `[attr.inputName]` — the ngx wrapper provides real Angular @Input()s.
