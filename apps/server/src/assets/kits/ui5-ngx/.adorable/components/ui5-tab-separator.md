# TabSeparator

**Type:** Component
**Selector:** `<ui5-tab-separator>`
**Import:** `import { TabSeparatorComponent } from '@ui5/webcomponents-ngx/main/tab-separator';`
**Export As:** `ui5TabSeparator`
**Package:** `@ui5/webcomponents` (main)

## Basic Usage
```html
<ui5-tab-separator></ui5-tab-separator>
```

## Description
The ui5-tab-separator represents a vertical line to separate tabs inside a ui5-tabcontainer.
