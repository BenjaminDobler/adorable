**Type:** Component

## Basic Usage
```html
<ui5-product-switch></ui5-product-switch>
```

## Description
The ui5-product-switch is an SAP Fiori specific web component that is used in ui5-shellbar and allows the user to easily switch between products. The ui5-product-switch provides advanced keyboard handling.
